Um pouco sobre mim

Me chamo Gabriel, sou nascido e criado na periferia do Rio de Janeiro. Minha paixão pela tecnologia me levou a conquistar uma bolsa de estudos em uma escola de excelência, onde fui apresentado a um mundo novo e inspirador. Tecnologias que já utilizei

As tecnologias abaixo são algumas das ferramentas que já utilizei enquanto construí meus projetos.

Tailwind

CSS Framework
Frontend

Tailwind CSS é um framework CSS open-source. O diferencial deste é que, ao contrário da grande maioria dos outros, ele não provê elementos já montados.

React

JS lib
Frontend

React é uma biblioteca front-end usada para criar interfaces baseadas em componentes. É atualmente mantida pela Meta.

Typescript

Linguagem de Programação
Adiciona tipagem estática ao JS.

TypeScript é uma linguagem de programação grátis e open-source, desenvolvida e mantida pela Microsoft.

Express.js

Node.js Framework
Backend

Express é um framework back-end para o Node.js, open-source e usado para construir aplicações WEB e APIs.
